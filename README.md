## 🚀 快速开始

### 1. 访问应用
直接在浏览器中打开应用，无需安装。

### 2. 登录/注册
- 使用手机号快速登录
- 支持社交账号登录 (可选)

### 3. 创作漫剧
**方式一：快速创作**
1. 进入「漫剧创作」页面
2. 点击「新建漫剧」
3. 输入创意描述
4. AI自动生成完整漫剧 (2-5分钟)

**方式二：单集创作**
1. 进入「短视频创作」页面
2. 输入故事描述
3. 选择视觉风格
4. 生成单集视频

### 4. 管理作品
- 查看所有创作的漫剧
- 编辑剧集和分镜
- 生成视频
- 下载和分享

## 🐛 问题排查

### 常见问题

**Q: 视频播放失败？**
A: 检查控制台错误，确认OSS视频URL是否正常

**Q: AI生成速度慢？**
A: 正常需要2-5分钟，请耐心等待

**Q: 登录失败？**
A: 检查手机号格式，确保络连接正常

**Q: 数据加载失败？**
A: 刷新页面，检查网络连接

### 调试工具
- 打开浏览器控制台 (F12)
- 查看Network标签 (网络请求)
- 查看Console标签 (错误日志)
